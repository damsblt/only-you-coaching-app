# Instructions pour vider le cache du navigateur

Si les nouvelles icônes SVG ne s'affichent pas, c'est probablement un problème de cache du navigateur.

## Méthodes pour vider le cache :

### Chrome / Edge
1. Appuyez sur `Ctrl+Shift+R` (Windows/Linux) ou `Cmd+Shift+R` (Mac) pour un rechargement forcé
2. Ou allez dans les DevTools (F12) > Onglet Network > Cochez "Disable cache"

### Firefox
1. Appuyez sur `Ctrl+Shift+R` (Windows/Linux) ou `Cmd+Shift+R` (Mac)
2. Ou allez dans les DevTools (F12) > Onglet Network > Cochez "Disable cache"

### Safari
1. Appuyez sur `Cmd+Option+R` pour un rechargement forcé
2. Ou allez dans Développement > Vider les caches

## Vérification directe

Vous pouvez aussi ouvrir directement une URL SVG dans le navigateur pour vérifier :
- https://only-you-coaching.s3.eu-north-1.amazonaws.com/thumbnails/audios/audio-1761060260169-ipea57g72.svg

Si l'icône s'affiche correctement en bleu (#39334D), alors c'est bien un problème de cache de la page web.
